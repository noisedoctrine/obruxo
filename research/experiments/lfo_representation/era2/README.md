# LFO Era 2

This is the clean workspace for the next LFO representation research phase.

Era 2 starts from the design contract in
[design/LFO_ERA2_DESIGN.md](./design/LFO_ERA2_DESIGN.md). The key point is that
topology may help offline codebook construction, but topology must not select
atoms after the codebook exists.

## Workspace

- [design/](./design/): design docs and settled conceptual contracts.
- [experiments/](./experiments/): future Era 2 experiment folders.
- [reports/](./reports/): future Era 2 result writeups.
- [artifacts/](./artifacts/): future generated outputs.
- [notes/](./notes/): working notes and sketches.

## Intentional Non-Scaffolding

No Era 2 code package has been created here. That is intentional. The first
implementation should be designed from the Era 2 problem framing, not copied
from the Era 1 module layout.
