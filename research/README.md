# Research

This directory contains OBRUXO's current research framing and experimental work.

- [`PROJECT_BRIEF.md`](PROJECT_BRIEF.md) defines the original problem.
- [`modelling/`](modelling/) contains the current architecture, research tracker, and dataset/curriculum strategy.
- [`vital/`](vital/) documents Vital's preset representation.
- [`experiments/`](experiments/) contains uncertain approaches being investigated; an experiment is not the identity of the overall project.
- [`archive/`](archive/) preserves historical research inputs that are no longer authoritative.
